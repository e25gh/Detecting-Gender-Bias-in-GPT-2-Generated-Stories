# How to Contribute  

## 🐛 Reporting Bias  
- Found a biased output? Open an Issue with:  
  - The prompt used  
  - The generated text  
  - Suggested improvements  

## 💡 Improving the Project  
- Add more professions to test (e.g., "The pilot...").  
- Suggest better metrics (e.g., using NLTK for sentiment analysis).
